## 2.1 Largest Element in Array

Given, array DATA of N elements
Find the LOC of the largest element of array

```
1. Start
2. Set MAX = DATA[0]
3. For K = 1 to n - 1, Repeat
        if DATA[K] > MAX, then
            Set MAX = DATA[K]
            Set LOC = K
4. Print LOC, MAX
5. Exit
```
